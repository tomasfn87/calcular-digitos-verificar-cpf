# Calcular Dígitos CPF

## Como usar

### Compilar

```console
javac Cpf.java
```

### Executar

#### Calcular dígitos verificadores CPF

```console
java Cpf -c 123.456.789
java Cpf --calcular 987.654.321
```

#### Verificar CPF

```console
java Cpf -v 123.456.789-09
java Cpf --verificar 987.654.321-09
```
